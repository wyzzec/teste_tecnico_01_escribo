**Baixe uma IDE compátivel com o dart, por exemplo, o Intellij. Após isso, baixe a versão 2.16.1 do SDK Dart e configure sua IDE. Depois, é só ir até "file/open...", abrir o projeto e rodar na sua IDE.**
